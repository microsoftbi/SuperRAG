## 二、测试场景

### 场景 1: NL2SQL 问数功能 (P0)

| # | 步骤 | 预期结果 | 状态 |
|:--:|---|---|:--:|
| 1.1 | 点击「问数」Tab | 切换到 NL2SQL 对话界面 | ⬜ |
| 1.2 | 输入"查询2025年各区域销售额" | 显示 SQL + 结果表格(含序号列、千分位数字) | ⬜ |
| 1.3 | 点击表格右上角「📊 生成图表」 | 弹出图表配置面板 | ⬜ |
| 1.4 | 选择「饼图」+ X=区域 + Y=销售额 → 点应用 | 表格下方渲染饼图 | ⬜ |
| 1.5 | 在饼图顶部下拉切换到「柱图」 | 图表切换为柱图 | ⬜ |
| 1.6 | 点「⬇️ PNG」 | 下载图表为 PNG 文件 | ⬜ |
| 1.7 | 输入"把刚才数据做成柱图" | LLM 自动调 make_chart,图表渲染在表格下方 | ⬜ |
| 1.8 | 输入"换成饼图" | LLM 调 make_chart,图表切换为饼图 | ⬜ |
| 1.9 | 点击表格右上角「📋 复制CSV」 | 表格数据以 CSV 格式复制到剪贴板 | ⬜ |
| 1.10 | 发送问数查询后,回到 RAG Tab | RAG 聊天不受影响,历史独立 | ⬜ |

## 三、边界场景

| # | 场景 | 预期 |
|:--:|---|---|
| B1 | 未配置 NL2SQL 数据库连接时问数 | 提示"未配置数据库连接" |
| B2 | 知识图谱无数据时打开 | 显示占位提示 |
| B3 | 不登录直接访问任何页面 | 重定向到登录页 |
| B4 | 非管理员访问管理后台 | 无权限提示 |
| B5 | 上传非常大的文档(>10MB) | 正常处理,不分块超时 |
| B6 | 空文件名上传 | 被 Pydantic 校验拦截(422) |
